Steve Segall — Kubernetes EKS Demo

This page is deployed using a fully containerized workflow on AWS Elastic Kubernetes Service (EKS).

What I Did

  • 🔧 Built a static web page using HTML and CSS
  • 🐳 Created a Dockerfile that serves the page using nginx
  • 📦 Pushed the Docker image to Docker Hub
  • ☸️ Wrote Kubernetes manifests:
    • deployment.yaml to run the Docker container on EKS
    • service.yaml to expose it via a LoadBalancer
  • ☁️ Set up an EKS cluster using eksctl and kubectl
  • 🌐 Pointed a custom domain to the AWS LoadBalancer
